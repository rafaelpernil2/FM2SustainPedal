Sustain pedal support for volca FM2

A web application to give sustain pedal support to the KORG volca FM2

This software is no longer needed. KORG has released a firmware with this feature. Update to firmware 1.10

Select your MIDI keyboard and channel as input, pick your volca FM2 as output, choose a different channel for input and output and have fun!
